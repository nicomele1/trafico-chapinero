# Data

This directory is for local input data and generated artifacts.

The repository may include small files needed for review, such as `data/raw/chapinero_road_network.png`. Larger generated files, including GraphML graphs, pickle files, notebook caches, and processed outputs, should remain local unless there is a specific reproducibility reason to commit them.

The road graph used by the notebooks is generated by `notebooks/01_download_graph.ipynb`, which uses OSMnx to query OpenStreetMap data available at execution time. Later notebooks may write intermediate results under `data/processed/`.

Because OpenStreetMap changes over time, a graph downloaded later may not be identical to a graph downloaded earlier.
